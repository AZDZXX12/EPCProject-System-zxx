## 系统概述

### 什么是EPC项目管理系统？

EPC项目管理系统是针对工程总承包（Engineering-Procurement-Construction）项目的全生命周期数字化管理平台。系统覆盖从项目设计、设备采购到施工建设的完整流程，实现项目信息化、可视化、智能化管理。

### 适用领域

- ⚗️ **化工工程** - 化工装置建设、改造项目
- 🏭 **石油化工** - 炼油、乙烯、化纤项目
- ⚡ **电力工程** - 火电、核电、新能源项目
- 🏗️ **冶金建设** - 钢铁、有色金属项目
- 🏛️ **工业建筑** - 厂房、仓储设施建设

### 核心价值

1. **全流程管理** - 覆盖EPC全生命周期
2. **数据集成** - 打破信息孤岛
3. **决策支持** - 数据分析与预警
4. **协同办公** - 多角色协作
5. **成本控制** - 实时成本监控

## EPC模式介绍

### 什么是EPC？

**EPC** = Engineering（工程设计）+ Procurement（设备采购）+ Construction（施工建设）

EPC是国际通行的工程总承包模式，承包商对工程的设计、采购、施工全过程负责，对工程的质量、安全、工期和造价全面负责。

### EPC模式特点

#### 1. **Engineering（工程设计）**
- 概念设计
- 基础设计
- 详细设计
- 设计变更管理
- 技术方案优化

#### 2. **Procurement（设备采购）**
- 设备选型
- 供应商管理
- 采购合同管理
- 设备到货验收
- 采购进度跟踪

#### 3. **Construction（施工建设）**
- 施工组织设计
- 进度计划管理
- 设备安装调试
- 质量安全控制
- 竣工验收

### EPC模式优势

✅ **责任主体明确** - 总承包商全面负责  
✅ **界面管理简化** - 减少协调成本  
✅ **风险可控** - 合同风险转移  
✅ **工期保证** - 一体化管理  
✅ **成本透明** - 总价合同模式

### 3. P - 设备采购管理

#### 3.1 设备选型采购 (P)

##### 功能说明
智能设备选型系统集成了Luckysheet表格引擎，提供类似Excel的专业选型工具。

##### 支持的设备类型

**1. 风机选型工具** 🌀
- **输入参数**:
  - 风量 (m³/h)
  - 风压 (Pa)
  - 工作温度 (℃)
  - 介质类型
- **输出结果**:
  - 推荐风机型号
  - 功率和转速
  - 噪音和效率
  - 价格预估

**2. 泵类设备选型** ⚙️
- **输入参数**:
  - 流量 (m³/h)
  - 扬程 (m)
  - 介质密度
  - 工作压力
- **输出结果**:
  - 推荐泵型号
  - 功率和效率
  - 材质选择
  - 价格预估

**3. 换热器选型** 🔥
- **输入参数**:
  - 换热量 (kW)
  - 换热面积 (m²)
  - 介质温度
  - 压力等级
- **输出结果**:
  - 推荐换热器型号
  - 传热系数
  - 结构参数
  - 价格预估

##### 选型操作流程

**Step 1: 选择设备类型**
```
进入"设备选型采购"模块 → 选择设备类别
```

**Step 2: 输入工况参数**
```
填写流量、压力、温度等工艺参数
```

**Step 3: 系统自动计算**
```
系统根据数据库匹配合适型号
```

**Step 4: 查看推荐结果**
```
查看多个推荐方案及技术参数
```

**Step 5: 导出选型报告**
```
生成Excel选型报告供审核
```

##### 数据导入导出

**导入Excel数据**:
1. 准备Excel文件（设备清单）
2. 点击"导入数据"按钮
3. 选择Excel文件
4. 系统自动解析数据
5. 确认导入

**导出Excel报表**:
1. 完成设备选型
2. 点击"导出数据"按钮
3. 选择导出格式（Excel/PDF）
4. 下载选型报告

##### 在线编辑功能
- 类Excel表格界面

### 4. C - 施工建设管理

#### 4.1 施工进度甘特图 (C)

##### 功能说明
可视化的施工进度管理工具，基于甘特图实现任务调度和进度监控。

##### 核心功能

**1. 任务创建与管理**
- 任务名称和描述
- 开始和结束日期
- 任务负责人
- 优先级设定
- 完成百分比

**2. 任务依赖关系**
```
前置任务 → 后续任务

示例:
基础施工 → 设备吊装 → 管道安装 → 调试验收
```

- **FS (Finish to Start)** - 前置任务完成后开始
- **SS (Start to Start)** - 同时开始
- **FF (Finish to Finish)** - 同时完成
- **SF (Start to Finish)** - 前置开始后完成

**3. 关键路径分析**
```
Critical Path Method (CPM)
自动识别影响总工期的关键任务链
```

- 关键任务标红显示
- 松弛时间计算
- 工期优化建议

**4. 进度偏差预警**

```
安全巡检（每天）:
□ 作业人员着装
□ 安全防护措施
□ 机械设备状态
□ 施工现场整洁

质量检查（每周）:
□ 焊缝质量
□ 安装精度
□ 材料验收
□ 隐蔽工程
```

#### 专项检查
```
月度检查:
- 综合安全检查
- 质量体系审核
- 文明施工检查
- 环境保护检查

季度检查:
- 设备维护保养
- 应急预案演练
- 证书有效期检查
- 培训效果评估
```

### 4. 数据录入规范

#### 进度数据
- **每日更新** - 施工进度和工作量
- **及时准确** - 当日数据当日录入
- **详细具体** - 包含具体工作内容

#### 质量数据
- **完整记录** - 检验项目、结果、结论
- **附件上传** - 照片、报告、证书
- **责任到人** - 检验人、复核人签字

#### 安全数据
- **实时上报** - 隐患和事故及时上报
- **闭环管理** - 整改措施和验收记录
- **统计分析** - 定期分析安全趋势

## 常见问题

### Q1: 如何快速创建项目？
**A**: 
1. 进入"项目管理 (E)"
2. 点击"新建项目"
3. 填写必填项（项目名称、编号、日期）
4. 保存即可

**建议**: 可以先创建项目模板，后续项目复制模板即可。

### Q2: 设备选型如何保证准确性？
**A**:
1. 确保输入参数准确（流量、压力、温度）
2. 核对介质特性
3. 考虑安全余量
4. 与供应商技术确认

**提示**: 选型结果仅供参考，最终需要供应商技术确认。

### Q3: 甘特图如何设置任务依赖？
**A**:
1. 创建前置任务和后续任务
2. 点击前置任务的连接点
3. 拖拽至后续任务
4. 选择依赖类型（FS/SS/FF/SF）

**示例**: 基础施工（前置）→ 设备安装（后续），类型：FS

### Q4: 如何批量导入设备数据？
**A**:
1. 下载Excel模板
2. 按模板格式填写设备信息
3. 在"设备选型采购"模块点击"导入"
4. 选择Excel文件上传
5. 确认导入

**注意**: Excel格式必须严格遵守模板要求。

### Q5: 进度数据如何导出？
**A**:
1. 进入"施工进度甘特图"
2. 点击"导出"按钮
3. 选择格式（PDF/Excel）
4. 下载文件

**用途**: 用于项目汇报、存档、打印。

### Q6: 如何管理设备安装验收？
**A**:
1. 进入"设备安装管理"
2. 找到目标设备
3. 点击"验收"按钮
4. 填写验收表单
5. 上传验收照片和报告
6. 提交验收记录

**流程**: 安装中 → 已安装 → 已验收

### Q7: 质量检测记录如何归档？
**A**:
1. 进入"HSE质量检测"
2. 创建检测记录
3. 上传检测报告PDF
4. 填写检测结果
5. 系统自动归档

**保存**: 所有记录永久保存，可随时查询。

### Q8: 如何进行多项目对比？
**A**:
1. 进入"EPC工程概览"
2. 查看项目列表
3. 对比进度百分比
4. 查看统计图表

**分析**: 识别进度滞后项目，及时采取措施。

### Q9: 系统数据如何备份？
**A**:
- 系统每日自动备份数据库
- 备份文件位于 `server/data/backup/`
- 可手动触发备份

**建议**: 定期下载备份文件到本地或网盘。

### Q10: 如何添加新用户？
**A**:
1. 管理员登录系统
2. 进入"用户管理"
3. 点击"新增用户"
4. 填写用户信息
5. 分配角色权限
6. 保存用户

**权限**: 管理员/项目经理/工程师/施工员
