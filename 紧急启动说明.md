# 🚨 紧急启动说明

## 问题诊断

您的系统**缺少Node.js环境**，导致无法启动前端服务器。

---

## ✅ 解决方案

### 方案1：安装Node.js（推荐）

1. **下载Node.js**
   - 访问：https://nodejs.org/
   - 下载：LTS版本（推荐 v18.x 或 v20.x）
   - 选择：Windows Installer (.msi) 64-bit

2. **安装Node.js**
   - 双击安装包
   - 一路点击"Next"
   - ✅ 确保勾选"Add to PATH"
   - 完成安装

3. **验证安装**
   ```cmd
   # 打开新的CMD窗口
   node --version
   npm --version
   ```

4. **启动前端**
   ```cmd
   cd C:\Users\Administrator\Desktop\xiangmu\client
   npm install  # 首次需要安装依赖
   npm start    # 启动开发服务器
   ```

---

### 方案2：使用VSCode内置终端

如果您正在使用VSCode：

1. 在VSCode中打开项目文件夹
2. 按 `Ctrl + ~` 打开终端
3. 执行：
   ```bash
   cd client
   npm start
   ```

---

### 方案3：检查已有Node.js

可能Node.js已安装但PATH未配置：

1. 搜索文件：在Windows搜索框输入 `node.exe`
2. 找到Node.js安装目录（通常在 `C:\Program Files\nodejs\`）
3. 将该目录添加到系统PATH环境变量
