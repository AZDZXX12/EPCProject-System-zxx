## 🔧 故障排查

### 问题1: 页面无法访问
**现象**: localhost拒绝连接

**解决方案**:
```bash
# 1. 检查端口是否被占用
netstat -ano | findstr "3000"

# 2. 手动启动前端
cd client
npm start

# 3. 手动启动后端
cd server
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 问题2: 甘特图显示异常
**现象**: 甘特图空白或错位

**解决方案**:
1. 强制刷新浏览器（Ctrl+Shift+R）
2. 清除浏览器缓存
3. 检查控制台是否有JavaScript错误
4. 确认任务数据已加载

### 问题3: 导出功能失败
**现象**: 导出PDF或Excel失败

**解决方案**:
1. 检查浏览器控制台错误信息
2. 确认任务数据已加载
3. 尝试重新加载页面
