┌───────────────────────────────────────────────────────────────────────┐
│  步骤 4.1：注册 Vercel 账号（1分钟）                                   │
└───────────────────────────────────────────────────────────────────────┘

1. 访问 Vercel 官网：
   🔗 https://vercel.com

2. 点击 "Sign Up" 注册账号

3. 强烈推荐：使用 GitHub 账号登录（最方便）
   ✅ 点击 "Continue with GitHub"
   ✅ 授权 Vercel 访问你的 GitHub
   ✅ 自动完成注册

   好处：
   - 自动关联仓库
   - 代码更新自动部署
   - 省去手动配置


┌───────────────────────────────────────────────────────────────────────┐
│  步骤 4.2：导入项目（2分钟）                                           │
└───────────────────────────────────────────────────────────────────────┘

1. 登录后，点击 "Add New..." → "Project"
   或者直接点击 "Add New Project"

2. 导入 Git 仓库：
   - 如果是第一次，点击 "Import Git Repository"
   - 在列表中找到: azdzxx/luckysheet-selection-system
   - 点击 "Import" 按钮

3. 配置项目设置：

   ┌─────────────────────────────────────────────────────────┐
   │ Project Name:                                           │
   │   luckysheet-selection-system                           │
   │   （可以自定义，这会成为你的域名）                       │
   │                                                         │
   │ Framework Preset:                                       │
   │   Other                                                 │
   │   （选择 "Other" 或 "None"）                            │
   │                                                         │
   │ Root Directory:                                         │
   │   ./  （留空，使用根目录）                              │
   │                                                         │
   │ Build and Output Settings:                             │
   │                                                         │
   │   Build Command:                                        │
   │     留空（不需要构建）                                   │
   │                                                         │
   │   Output Directory:                                     │
   │     dist-refactored                                     │
   │     ⚠️ 重要：必须填写这个！                              │
   │                                                         │
   │   Install Command:                                      │
   │     留空（静态文件，不需要安装依赖）                     │
   └─────────────────────────────────────────────────────────┘

4. 环境变量（Environment Variables）：
   对于这个项目，不需要添加环境变量
   直接跳过即可

5. 点击 "Deploy" 按钮

6. 等待部署完成（约1-2分钟）
   - 你会看到部署进度
   - 部署成功会显示庆祝动画 🎉


┌───────────────────────────────────────────────────────────────────────┐

   📋 https://luckysheet-selection-system.vercel.app
   （实际地址可能不同，取决于你的项目名称）

2. 点击 "Visit" 或直接访问URL

3. 测试前端功能：

   ✅ 可以看到 "设备参数选型系统" 标题
   ✅ 工具栏按钮显示正常
   ✅ 可以点击 "新建" 创建空白表格

4. 记录你的前端URL：

   💡 Vercel 部署提示
═══════════════════════════════════════════════════════════════════════════

1. 自动部署：
   每次推送代码到 GitHub，Vercel 会自动重新部署

2. 自定义域名（可选）：
   Settings → Domains → 添加你自己的域名
   免费且自动配置 HTTPS

3. 部署历史：
   Deployments 标签可以查看所有部署记录
   可以回滚到任意历史版本

4. 性能分析：
   Analytics 标签可以查看访问统计（免费）
