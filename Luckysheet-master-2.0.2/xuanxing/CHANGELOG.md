# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵守 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-01

### 新增
- 电缆参数输入表单
- 基于IEC标准的电缆计算引擎
  - IEC 60287-1-1 载流量计算
  - IEC 60512 温度校正
  - IEC 60364-5-52 敷设方式校正
- 电压降计算功能
- 3D电缆结构可视化
- 多方案对比和推荐
- RESTful API接口
- PostgreSQL数据库集成
- Docker容器化部署
- 完整的单元测试

### 数据库
- 电缆规格数据表
- 预置12种常用电缆规格
- 支持YJV、VV、YJV22等型号

### 前端功能
- Material-UI响应式界面
- Three.js 3D渲染
- 实时计算和结果展示
- 参数验证和错误提示

### 后端功能
- Django REST Framework API
- 电缆规格管理
- 计算引擎服务
- 数据导入导出

### 文档
- 完整的README文档
- API接口文档
- 贡献指南
- Docker部署指南
