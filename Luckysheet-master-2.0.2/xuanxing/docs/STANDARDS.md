# 技术标准说明

本文档详细说明电缆选型工具所依据的国际和国家技术标准。

## 国际标准（IEC）

### IEC 60287-1-1: 电缆载流量计算

**标准全称**: Electric cables - Calculation of the current rating - Part 1-1: Current rating equations (100 % load factor) and calculation of losses - General

**应用范围**: 
- 计算电缆在恒定负载下的额定载流量
- 考虑电缆损耗和散热

**核心公式**:
```
I = √[ΔT / (R × Rt × (1 + λ1 + λ2))]
```

其中：
- I: 载流量 (A)
- ΔT: 允许温升 (K)
- R: 导体交流电阻 (Ω/m)
- Rt: 热阻 (K·m/W)
- λ1: 屏蔽损耗系数
- λ2: 铠装损耗系数

**本系统实现**: `backend/cables/calculators.py` 中的 `calculate_cross_section()` 函数

---

### IEC 60512: 连接器和机械开关用固定的电气连接

**标准全称**: Connectors for electronic equipment - Tests and measurements

**温度校正系数公式**:
```
k_temp = √[(T_max - T_ambient) / (T_max - T_base)]
```

其中：
- T_max: 最高工作温度 (90℃ for XLPE)
- T_ambient: 环境温度 (℃)
- T_base: 基准温度 (30℃)

**温度对照表**:

| 环境温度 | XLPE绝缘 | PVC绝缘 |
|----------|----------|---------|
| 20℃ | 1.10 | 1.12 |
| 30℃ | 1.00 | 1.00 |
| 40℃ | 0.91 | 0.87 |
| 50℃ | 0.82 | 0.71 |

**本系统实现**: 
- `backend/cables/calculators.py` 中的 `get_temp_correction()`
- `frontend/src/utils/currentCorrection.ts` 中的 `getTempCorrection()`

### IEC 60364-5-52: 低压电气装置 - 第5-52部分：电气设备的选择和安装 - 布线系统

**标准全称**: Low-voltage electrical installations - Part 5-52: Selection and erection of electrical equipment - Wiring systems

**敷设方式校正系数**:

| 敷设方式 | 校正系数 | 说明 |
|----------|----------|------|
| 空气中单根 | 1.00 | 理想散热条件 |
| 桥架敷设 | 0.95 | 良好通风 |
| 管道敷设 | 0.80 | 散热受限 |
| 直埋敷设 | 1.00 | 土壤散热 |
| 多根并列 (n>3) | 0.8 + 0.2/√n | 互相影响 |

**本系统实现**: `backend/cables/calculators.py` 中的 `get_grouping_correction()`

## 中国国家标准（GB）

### GB/T 50217-2018: 电力工程电缆设计标准

**应用范围**:
- 电力工程中电缆的选择、敷设和保护
- 电缆载流量计算
- 电压降计算

**电压降限制**:
- 照明线路: ≤3%
- 动力线路: ≤5%
- 特殊设备: 按设备要求

**电压降计算公式**:
```
ΔU% = (√3 × I × L × (R×cosφ + X×sinφ)) / U × 100
```

其中：
- I: 负载电流 (A)
- L: 线路长度 (km)
- R: 电阻 (Ω/km)
- X: 电抗 (Ω/km)
- cosφ: 功率因数
- U: 额定电压 (V)

**本系统实现**: `backend/cables/calculators.py` 中的 `calculate_voltage_drop()`

## 计算示例

### 示例1: 标准条件下的计算

**输入参数**:
- 额定电压: 380V
- 负载电流: 100A
- 线路长度: 50m
- 环境温度: 30℃
- 敷设方式: 桥架

**计算过程**:

1. 温度校正系数:
   ```
   k_temp = √[(90-30)/(90-30)] = 1.0
   ```

2. 敷设校正系数:
   ```
   k_group = 0.95
   ```

3. 所需额定载流量:
   ```
   I_rated = 100 / (1.0 × 0.95) = 105.3A
   ```

4. 选择电缆: YJV 4×25mm² (额定载流量115A)

5. 电压降:
   ```
   R = 0.727 / 25 = 0.029 Ω/km
   ΔU% = (1.732 × 100 × 0.05 × 0.029 × 0.85) / 380 × 100
       = 0.92%
   ```

**结论**: YJV 4×25mm² 满足要求

---

### 示例2: 高温环境计算

**输入参数**:
- 额定电压: 380V
- 负载电流: 100A
- 线路长度: 50m
- 环境温度: 40℃
- 敷设方式: 管道

**计算过程**:

1. 温度校正系数:
   ```
   k_temp = √[(90-40)/(90-30)] = 0.91
   ```

2. 敷设校正系数:
   ```
   k_group = 0.80
   ```

3. 所需额定载流量:
   ```
   I_rated = 100 / (0.91 × 0.80) = 137.4A
   ```

4. 选择电缆: YJV 4×35mm² (额定载流量145A)

**结论**: 高温和管道敷设需要更大截面
