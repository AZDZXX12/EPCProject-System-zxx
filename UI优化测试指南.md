## 🐛 已知问题

### 非功能性警告（可忽略）

**Console 中的警告：**
```
CSS inline styles should not be used
```

**说明：**
- 这些是 ESLint 的样式警告
- 不影响功能和性能
- 动态样式（如进度条、根据状态变化的颜色）必须使用内联样式
- 可以安全忽略
