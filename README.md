# 化工项目管理系统

🚀 **一个功能完善的化工设备安装项目管理系统**

集成数字孪生、甘特图、设备管理、安全质量监控等功能，具有赛博朋克科技风格的现代化UI设计。

---

- Glitch故障艺术效果
- 实时系统时钟

## 🛠 技术栈

### 前端
- **React 18** + **TypeScript**
- **Ant Design** - UI组件库
- **React Three Fiber** - 3D渲染
- **Canvas API** - 甘特图绘制
- **React Router** - 路由管理
- **Axios** - HTTP请求
- **Day.js** - 日期处理
- **jsPDF** + **html2canvas** - PDF导出

### 后端
- **FastAPI** - Python Web框架
- **Uvicorn** - ASGI服务器
- **SQLAlchemy** - ORM
- **Pydantic** - 数据验证
- **Python-dotenv** - 环境变量管理

## 🚀 快速启动

### 本地运行

#### 方法1：一键启动（推荐）

**Windows**
```bash
# 双击运行
启动.bat

# 或使用PowerShell
.\启动.ps1
```

#### 方法2：手动启动

**后端**
```bash
cd server
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**前端**
```bash
cd client
npm install
npm start
```
