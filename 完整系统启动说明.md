## 📝 注意事项

3. 占用端口：3000, 8000

1. 无需React服务器

4. 功能独立，不依赖其他服务

## 🔧 故障排查

### Q1: 完整系统启动后是空白？
**答**: React服务器可能未就绪

**解决**:
1. 等待30秒
2. 按F5刷新Electron窗口
3. 或在浏览器访问 http://localhost:3000 确认React是否启动

### Q2: 提示端口被占用？
**答**: React或后端已在运行

**解决**:
1. 关闭所有相关窗口
2. 或手动关闭进程：
```bash
# 查找进程
netstat -ano | findstr :3000
# 关闭进程
taskkill /F /PID [PID]
```

### Q3: 设备选型打不开？
**答**: 选型系统文件路径问题

**解决**:
1. 确认文件存在：`client\public\selection-tools\`
2. 检查是否有XLSX文件：`libs\xlsx.full.min.js`
