# 设备采购管理系统使用说明

## 📋 系统概述

**设备采购管理 (P)** 是EPC项目管理系统中负责设备采购全流程管理的核心模块。

### 功能定位
- **位置**: 主菜单 "设备采购管理 (P)"（项目管理下方）
- **作用**: 管理整个项目的设备清单、厂家信息、报价、交货期等
- **联动**: 与"设备选型系统"无缝集成

---

## 🎯 核心功能

### 1. 设备清单管理

#### 统计看板
```
┌────────────┬────────────┬────────────┬────────────┐
│ 设备总数   │ 采购总额   │ 已到货     │ 运输中     │
│ 150件      │ ¥5,280万   │ 45/150    │ 30/150    │
└────────────┴────────────┴────────────┴────────────┘
```

#### 设备信息字段
| 字段 | 说明 | 示例 |
|-----|------|------|
| 设备编号 | 唯一标识 | EQ-001 |
| 设备名称 | 设备类型 | 离心泵 |
| 规格型号 | 具体型号 | IS80-50-200 |
| 技术规格 | 性能参数 | 流量50m³/h，扬程32m |
| 数量 | 采购数量 | 3台 |
| 制造厂家 | 生产厂商 | 上海水泵厂 |
| 联系人/电话 | 厂家联系方式 | 张经理 / 021-12345678 |
| 单价/总价 | 价格信息 | ¥12,500 / ¥37,500 |
| 交货期 | 预计到货时间 | 2024-02-15 |
| 状态 | 采购状态 | 已到货 |

---

### 2. 采购状态管理

#### 状态流转
```
待询价 → 已报价 → 已下单 → 已到货
  ⬜      🔵       🟡       🟢
```

#### 状态说明
- **待询价**: 新增设备，尚未询价
- **已报价**: 已获得厂家报价
- **已下单**: 已签订采购合同
- **已到货**: 设备已送达现场

---

### 3. 与选型系统联动

#### 导入流程
```
设备选型系统 → 选型完成 → 导出清单 → 导入采购管理
    ⬇️
(技术参数选型)
    ⬇️
(生成设备清单)
    ⬇️
(一键导入)
    ⬇️
(完善厂家报价)
```

#### 操作步骤
1. 在"设备选型系统"中完成参数选型
2. 在"设备采购管理"页面点击【从选型系统导入】
3. 系统自动导入设备清单（设备名称、型号、规格、数量）
4. 手动补充厂家、价格、交货期等采购信息

## 🚀 使用指南

### 场景1: 新增设备采购

**步骤**:
```
1. 点击【添加设备】按钮
2. 填写设备信息（必填项：名称、型号、规格、数量、单位）
3. 填写厂家信息（制造厂家、联系人、电话）
4. 填写价格信息（单价，系统自动计算总价）
5. 选择状态（默认：待询价）
6. 点击【确定】保存
```

**表单示例**:
```yaml
设备名称: 离心泵
规格型号: IS80-50-200
技术规格: 流量50m³/h，扬程32m，功率11kW
数量: 3
单位: 台
制造厂家: 上海水泵厂
联系人: 张经理
联系电话: 021-12345678
单价: 12500元
交货期: 2024-02-15
状态: 待询价
```

---

### 场景2: 从选型系统导入

**适用情况**: 已在选型系统中完成设备选型

**操作**:
```
1. 进入"设备采购管理"页面
2. 点击工具栏【从选型系统导入】按钮
3. 确认导入（现有数据保留）
4. 系统导入设备清单（基础信息）
5. 编辑导入的设备，补充厂家和报价信息
```

**导入内容**:
- ✅ 设备名称
- ✅ 规格型号
- ✅ 技术规格
- ✅ 数量和单位
- ❌ 厂家信息（需手动补充）
- ❌ 价格信息（需手动补充）
- ❌ 交货期（需手动补充）

---

### 场景3: 编辑设备信息

**常见编辑操作**:
```
更新报价: 修改单价和总价
更新状态: 待询价 → 已报价 → 已下单 → 已到货
更新交货期: 修改预计到货日期
更新厂家: 更换供应商
添加备注: 记录特殊要求或注意事项
```

**操作**:
```
1. 找到要编辑的设备行
2. 点击【编辑】按钮
3. 修改相关信息
4. 点击【确定】保存
```

---

### 场景4: 导出采购清单

**用途**: 
- 向采购部门提供采购清单
- 向领导汇报采购进度
- 存档备查

**操作**:
```
1. 点击工具栏【导出Excel】按钮
2. 系统自动生成Excel文件
3. 文件名格式：设备采购清单_项目名称_日期.xlsx
4. 自动下载到本地
```

**导出内容**:
- 完整设备信息（14个字段）
- 当前项目所有设备
- 中文表头，易于阅读

## 🆘 常见问题

### Q1: 如何从选型系统导入设备？
**A**: 点击【从选型系统导入】按钮，系统会自动将选型系统中的设备清单导入到采购管理中。导入后需要手动补充厂家、价格等商务信息。

### Q2: 导入后原有数据会丢失吗？
**A**: 不会。导入操作是追加模式，原有数据会被保留，新导入的数据会添加到列表末尾。

### Q3: 如何修改已到货设备的信息？
**A**: 点击设备行的【编辑】按钮，修改相关信息后保存即可。所有状态的设备都可以编辑。

### Q4: 总价是如何计算的？
**A**: 总价 = 单价 × 数量。在填写单价和数量后，系统会自动计算总价。

### Q5: 导出的Excel文件在哪里？
**A**: 导出的文件会自动下载到浏览器默认下载目录（通常是"下载"文件夹）。
